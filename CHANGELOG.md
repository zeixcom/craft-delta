# Changelog

## 1.0.0 - 2026-03-02

### Added
- Revision comparison with side-by-side diff view
- Field-level diffing for all supported field types
- Word-level text diff with inline red/green highlighting
- Slideout UI triggered from revision breadcrumb
- Revision selector dropdowns
- Changed fields filter
- Diff summary stats
- Support for Plain Text, CKEditor, Matrix, relational, option, table, and scalar fields
- Third-party field differ registration via event
- Dark mode support
- German translation
